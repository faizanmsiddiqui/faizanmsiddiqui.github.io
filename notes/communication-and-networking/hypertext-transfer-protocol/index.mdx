# Hypertext Transfer Protocol (HTTP)

The Hypertext Transfer Protocol (HTTP) is the foundational protocol used for
transmitting data over the World Wide Web (WWW or W3). It defines how messages
are formatted and transmitted, and how web servers and browsers should respond
to various commands.
