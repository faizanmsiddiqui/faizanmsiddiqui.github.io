# Text Editors

- [**Kate**](https://kate-editor.org/): An advanced, cross-platform programming
  text editor with all the features included.
- [**Neovim**](https://neovim.io): A hyperextensible, Vim-based text editor that
  improves upon Vim's extensibility and usability.
- [**Notepad++**](https://notepad-plus-plus.org/): A free and open-source code
  editor and Windows Notepad replacement that supports several languages.
- [**Sublime Text**](https://www.sublimetext.com/): A sophisticated text editor
  for code, markup, and prose, known for its speed and efficiency.
- [**Vim**](https://www.vim.org/): A highly configurable, terminal-based text
  editor built for efficient text editing.
- [**Visual Studio Code**](https://code.visualstudio.com/): A free code editor,
  redefined and optimized for building and debugging modern applications, with
  support for a wide range of programming languages and frameworks.
