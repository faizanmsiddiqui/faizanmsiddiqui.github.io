# Emojis

Emojis are a fun and expressive way to enhance communication in digital spaces.
Below is a collection of popular emoji sets that can be used in various
contexts:

- [**Fluent Emoji**](https://github.com/microsoft/fluentui-emoji): A collection
  of emojis designed by Microsoft, optimized for use in Fluent UI applications.
- [**Noto Emoji**](https://github.com/googlefonts/noto-emoji): A comprehensive
  emoji font from Google that supports a wide range of emojis across different
  platforms.
- [**Streamline Emojis**](https://streamlinehq.com/emojis): A set of emojis
  designed for use in Streamline's design system, offering a unique style and
  extensive coverage of common emojis.
- [**Twemoji**](https://github.com/twitter/twemoji): X's (formerly Twitter)
  open-source emoji project, providing high-quality emojis that are consistent
  across platforms.
